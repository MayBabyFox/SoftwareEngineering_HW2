__Название и краткое описание проекта__  
Консольное приложение, которое позволяет создавать блюда, добавлять их в "рецепты", масштабировать порции и генерировать список покупок. 

__Использование__  
- Запуск кода:
  1. Открыть файл 'main.py' в в любой среде разработки для Python (PyCharm, VS Code и др.)
  2. Нажать кнопку запуска
- Запуск тестов:
  1. Открыть файл 'test_recipes.py'
  2. Нажать на пустом месте правой кнопкой мыши
  3. Выбрать "Run..."
- Установка:
  1. Скачать проект с GitHub
  2. Распаковать папку
  3. Открыть папку
  4. Открыть терминал
  5. Установить pytest
  6. Запустить тесты
